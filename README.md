## Overview
Khademni is a multi-platform project connecting users and freelancers. This repository contains the **desktop version** built with **JavaFX**, part of the integrated web and mobile ecosystem.

## Tech Stack
- **Desktop:** JavaFX  
- **Database:** MySQL (shared across platforms)  
- **Other Platforms:** Web (Symfony), Mobile (Codename One)

## Key Features
- Connects users seeking services with freelancers efficiently  
- Supports search, filtering, and scheduling via Google Calendar  
- Automates email and SMS notifications  
- Handles complaints and improves navigation for a smooth user experience

## Achievements
- Enabled multi-platform consistency with a **shared database**  
- Delivered a functional desktop app that complements the web and mobile apps

- Implementing user notifications and interactive features in JavaFX
